# Dự án cuối kỳ môn Cấu trúc dữ liệu và giải thuật
## Đề tài số 12: Hệ thống Quản lý Chức năng "Hoàn tác/làm lại" (Undo/Redo)
Ứng dụng xây dựng giao diện WinForm mô phỏng một caculator cơ bản có chức năng Undo/Redo.  
- Ngôn ngữ: C# (.NET 8-9, WinForms)  
- IDE: Visual Studio Code  
- Nhóm: 7  
- Chức năng chính:
  - Thực hiện các phép tính cơ bản cộng, trừ, nhân, chia ;
  - Undo/Redo thao tác người dùng ; 
  - Giao diện thân thiện, tối giản, dễ sử dụng .
